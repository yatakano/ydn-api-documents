# AdStyle (enum)
AdStyle displays the ad style type.

### Service
+ [AdGroupAdService](../services/AdGroupAdService.md)

| Value | Description | 
|---|---|
| TEXT | Text ad style. |
| IMAGE | Image ad style. |
| ANIMATION_IMAGE | Animation image ad style. |
